BinPy.combi_logic.mux_demux module¶

class BinPy.combi_logic.mux_demux.DEMUX[source]¶

This class can be used to create DEMUX in your circuit. It has a method for each kind of DEMUX, namely, demux_1_2, demux_1_4, demux_1_8 and demux_1_16 All the methods take 3 parameters(inputs, select lines and strobe). Strobe is high by default inputs and select_inputs are lists. First index of input is ‘A’, second index is ‘B’ and so on... But last index of select_inputs is ‘S0’, second last index is ‘S1’ and so on...
